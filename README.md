# Network Scanner

A Python-based network scanner to detect open ports and running services on target hosts. Ideal for learning network security, penetration testing, and reconnaissance techniques.

## Features
- Multi-threaded scanning for faster results.
- Detects open ports across a range of targets.
- Logs results into a report file.
- Handles exceptions and timeouts to avoid crashes during scanning.
